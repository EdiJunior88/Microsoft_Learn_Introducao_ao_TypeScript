Laboratório – usar tipos no TypeScript

Neste laboratório, você escreverá o JavaScript em cada um dos exercícios abaixo no TypeScript, deixando cada uma das variáveis fortemente tipadas.

Exercício 1: modificar o código JavaScript existente para ter variáveis fortemente tipadas

  1. Clone o repositório inicial inserindo o conteúdo a seguir no prompt de comando.
    2. Bash git clone https://github.com/MicrosoftDocs/mslearn-typescript cd mslearn-typescript/code/module-02/m02-start code .
  2. Abra o arquivo module02.ts e localize Exercício 1.
  3. Modifique o código para adicionar tipos às declarações de variável. O código JavaScript compilado deve ter a mesma aparência do exemplo original quando você terminar.
    4. JavaScript /* EXERCISE 1 TODO: Modify the code to add types to the variable declarations. The resulting JavaScript should look the same as the original example when you're done. */ let firstName; let lastName; let fullName; let age; let ukCitizen; firstName = 'Rebecca'; lastName = 'Smith'; age = 42; ukCitizen = false; fullName = firstName + " " + lastName; if (ukCitizen) { console.log("My name is " + fullName + ", I'm " + age + ", and I'm a citizen of the United Kingdom."); } else { console.log("My name is " + fullName + ", I'm " + age + ", and I'm not a citizen of the United Kingdom."); }

Exercício 2: modificar o código JavaScript existente para garantir resultados operacionais usando variáveis fortemente tipadas

  1. Localize o Exercício 2 em module02.ts.
  2. Você pode usar tipos para garantir resultados de operações. Execute o código como está e modifique-o para ter variáveis fortemente tipadas.
  3. Resolva os erros encontrados para que o resultado retornado seja 12.
    4. JavaScript /* EXERCISE 2 TODO: Run the code as is and then modify it to have strongly typed variables. Then, address any errors you find so that the result returned to a is 12. */ let x; let y; let a; x = 'five'; y = 7; a = x + y; console.log(a);

Exercício 3: implementar um tipo enumerado

  1. Localize o Exercício 3 em module02.ts.
  2. Implemente um tipo enum chamado Season que represente as constantes "Fall", "Winter", "Spring" e "Summer".
  3. Atualize a função para passar a estação referenciando um item na enum, por exemplo Season.Fall, em vez da cadeia de caracteres literal "Fall".
    4. JavaScript /* EXERCISE 3 TODO: In the following code, implement an enum type called Season that represents the constants "Fall", "Winter", "Spring", and "Summer". Then, update the function so you can pass in the season by referencing an item in the enum, for example Season.Fall, instead of the literal string "Fall". */ function whichMonths(season) { let monthsInSeason: string; switch (season) { case "Fall": monthsInSeason = "September to November"; break; case "Winter": monthsInSeason = "December to February"; break; case "Spring": monthsInSeason = "March to May"; break; case "Summer": monthsInSeason = "June to August"; } return monthsInSeason; } console.log(whichMonths("Fall"));

Exercício 4: declarar um tipo de matriz

  1. Localize o Exercício 4 em module02.ts.
  2. Declare a matriz como o tipo para corresponder ao tipo dos itens na matriz.
    3. JavaScript /* EXERCISE 4 TODO: Declare the array as the type to match the type of the items in the array. */ let randomNumbers; let nextNumber; for (let i = 0; i < 10; i++) { nextNumber=Math.floor(Math.random() * (100 - 1)) + 1; randomNumbers.push(nextNumber); } console.log(randomNumbers);
