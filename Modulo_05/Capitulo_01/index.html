Introdução

Como um programador de JavaScript, você pode ter experiência com classes ou com a implementação anterior ao ECMAScript 2015 (ES6) baseada em construtores e protótipos. As classes JavaScript no ES6 são essencialmente um açúcar sintático em relação à herança baseada em protótipos existente do JavaScript.

As classes no TypeScript estendem a funcionalidade ES6 adicionando recursos específicos do TypeScript, como anotações de tipo para membros de classe, modificadores de acesso e a capacidade de especificar parâmetros obrigatórios ou opcionais. Outro benefício do TypeScript é que você pode usá-lo no desenvolvimento com classes e compilá-las para JavaScript, que funciona em todos os principais navegadores e plataformas, conforme necessário.

Objetivos de aprendizagem

Depois de concluir este módulo, você poderá:

  • Explicar como pode usar classes do TypeScript para descrever a forma de objetos.
  • Declarar uma classe usando o TypeScript.
  • Criar uma instância de uma classe usando o TypeScript.
  • Aplicar modificadores de acesso a uma classe.
  • Definir propriedades estáticas em uma classe.
  • Declarar uma classe que estenda outra classe.
  • Declarar uma interface para garantir a forma da classe.
  • Determinar quando usar uma interface ou uma classe para definir a estrutura de um objeto.
