Exercício: selecionar um editor de TypeScript

Você pode escrever código TypeScript em qualquer editor de texto ou IDE de sua escolha.

Neste exercício, você aprenderá sobre suas opções de editor de texto e configurará o Visual Studio Code para poder então concluir os exercícios restantes neste módulo.

O Playground do TypeScript

Anteriormente, você escreveu um código TypeScript simples no Playground. O Playground é um local onde você pode escrever, compartilhar e aprender sobre o TypeScript. Você pode usá-lo para:

  • Aprender a usar o TypeScript em um lugar onde nenhum dano é causado.
  • Fazer experiências com a sintaxe do TypeScript e compartilhar
  • as URLs com outras pessoas.
  • Experimentar os vários recursos do compilador do TypeScript.

Neste curso, você ocasionalmente visitará o Playground para experimentar novos conceitos do TypeScript.

Ambientes de desenvolvimento integrado

Para ferramentas e edição de TypeScript com recursos completos, você encontrará muitos IDEs (ambientes de desenvolvimento integrado) que dão suporte a TypeScript ou que têm plug-ins ou pacotes de ferramentas desenvolvidos para dar suporte a ele. Neste módulo usamos o Visual Studio Code, mas você pode usar qualquer editor que preferir.

Instalar o Visual Studio Code

O Visual Studio Code é um editor de código-fonte leve e gratuito que é executado em sua área de trabalho nos sistemas operacionais macOS, Linux e Windows. Ele tem suporte interno para JavaScript, TypeScript e Node.js, além de um ecossistema avançado de extensões para outras linguagens e runtimes.

Para instalar o Visual Studio Code:

  1. Vá para a página de download do Visual Studio Code e selecione a opção de download para o seu sistema operacional.
  2. Se precisar de diretrizes de instalação adicionais, visite a página de instalação do seu sistema operacional:
    • Linux
    • macOS
    • Windows

Agora que você tem um editor de código, você está pronto para instalar o TypeScript.
