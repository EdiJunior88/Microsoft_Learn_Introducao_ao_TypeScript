Introdução

Se você sabe escrever funções em JavaScript, também saberá escrever funções no TypeScript. Mas o TypeScript adiciona alguns recursos novos às funções JavaScript padrão para facilitar o trabalho com elas.

Objetivos de aprendizagem

Depois de concluir este módulo, você poderá:

  • Explicar os benefícios do uso de tipos em funções.
  • Escrever funções que têm parâmetros rest, obrigatórios, opcionais e padrão.
  • Definir os tipos de função usando aliases de tipo ou interfaces.

Pré-requisitos

  • Conhecimento de TypeScript
  • Familiaridade com JavaScript
  • Familiaridade com funções e matrizes em JavaScript
  • Software instalado:
    • Git
    • Visual Studio Code
    • Node.js
    • TypeScript – o TypeScript pode ser instalado após a instalação do Node.js, depois de executar o seguinte em um terminal ou em uma janela de terminal:
    Bash npm install -g typescript

Para obter melhores resultados, siga as instruções completas para configurar seu ambiente e usar o compilador do TypeScript no Módulo 01: Introdução ao TypeScript.
