Exercício – Trabalhar com parâmetros

Parâmetros padrão

  1. Insira a função de seta a seguir, que aceita três parâmetros obrigatórios.
    2. TypeScript let subtractThreeNumbers = (x: number, y: number, z: number): number => x - y - z;
  2. Atribua um valor padrão de 100 ao parâmetro z substituindo z: number por z = 100.
    3. TypeScript let subtractThreeNumbers = (x: number, y: number, z = 100): number => x - y - z;
  3. Tente chamar a função com dois e três argumentos para testar o resultado.
    4. TypeScript subtractThreeNumbers(10, 20); // returns -110 because 'z' has been assigned the value 100 subtractThreeNumbers(10, 20, 15); // returns -25
