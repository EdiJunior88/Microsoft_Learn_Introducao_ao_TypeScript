Exercício – Trabalhar com parâmetros

Neste exercício, você criará funções que têm parâmetros obrigatórios, opcionais e padrão.

Parâmetros obrigatórios

  1. Abra o Playground e remova qualquer código existente.
  2. Insira a função de seta a seguir, que aceita três parâmetros obrigatórios.
    3. TypeScript let addThreeNumbers = (x: number, y: number, z: number): number => x + y + z;
  3. Tente chamar a função digitando addThreeNumbers(10, 20). O TypeScript gera o erro Eram esperados três argumentos, mas foram obtidos dois. Um argumento para 'z' não foi fornecido. Quando executada, a função retorna NaN porque o terceiro argumento foi passado como undefined, invalidando o cálculo.
  4. O que acontece ao digitar addThreeNumbers(10, 20, 30, 40)? O TypeScript gera o erro Eram esperados três argumentos, mas foram obtidos quatro. Quando é executada, a função remove o quarto e retorna 60.

Parâmetros opcionais

  1. Na função, tente tornar o parâmetro y opcional. O que acontece?
    2. TypeScript let addThreeNumbers = (x: number, y?: number, z: number): number => x + y + z;
  2. O TypeScript gera um erro porque a posição dos parâmetros opcionais é importante. Na lista de parâmetros, os parâmetros opcionais devem seguir todos os parâmetros obrigatórios. Em vez do parâmetro y, tente tornar o parâmetro z opcional. Além disso, para que essa função retorne o valor correto, você também deve atualizá-la para lidar com a possibilidade de que z já tenha sido transmitido como undefined. Agora você deve conseguir chamar a função usando addThreeNumbers(10, 20) ou addThreeNumbers(10, 20, 30).
    3. TypeScript let addThreeNumbers = (x: number, y: number, z?: number): number => { if((z === undefined)) { return x + y; } else { return x + y + z; } };
